Appointment Booking System

Windows Forms App 路 PostgreSQL 路 C#

Project Overview

This is a desktop-based C# Windows Forms application that allows users to schedule and view medical appointments. Built using Visual Studio and connected to a PostgreSQL database using Npgsql, the app supports real-time form validation, time and date selection, and displays booked appointments in a dynamic DataGridView.

Tech Stack: C#, .NET, Windows Forms, PostgreSQL, Npgsql
